Manipulate Date

We can create a Date object by using sytax:

var now = new Date();

Eg: Calculate how many days until Xmas

            var now  = new Date()
            var xmas = new Date(2019, 11, 25)
            
            var milliseconds = xmas - now
            
            var seconds      = milliseconds / 1000
            var mins         = seconds      / 60
            var hours        = mins         / 60
            var days         = hours        / 24
            
            alert(days.toFixed(2)  + " days until xmas")
            alert(Math.round(days) + " days until xmas")
